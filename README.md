# NetGuardian

## Sobre o projeto

NetGuardian é uma aplicação criada para facilitar o monitoramento de dispositivos de rede, oferecendo um sistema simples, seguro e intuitivo para verificar conectividade, armazenar logs e gerenciar um inventário de dispositivos em um único lugar.

Em um cenário onde a estabilidade de rede é essencial, o NetGuardian surge como uma ferramenta acessível e eficiente. O usuário pode cadastrar dispositivos informando:

* Nome do dispositivo
* Endereço IP
* Localização
* Descrição (opcional)

Após o cadastro, ele tem acesso a um menu que oferece:

* **Teste de conectividade (Ping)** – verificação instantânea do status do dispositivo
* **Consulta de logs** – histórico de disponibilidade e resultados dos testes
* **Inventário completo** – visualização organizada de todos os dispositivos cadastrados

---

## Tecnologias utilizadas

### Back-end

* Node.js
* Express
* MySQL / PostgreSQL / SQLite
* Comunicação via API REST

### Funcionalidades principais

* Cadastro de dispositivos
* Teste de conectividade via ping
* Armazenamento de logs no banco de dados
* Consulta via interface web
* Navegação simples e intuitiva

---

## Objetivo do projeto

O NetGuardian tem como objetivo tornar o monitoramento de dispositivos de rede mais acessível, centralizado e eficaz, permitindo um acompanhamento claro e direto da infraestrutura, auxiliando equipes de TI e ambientes que dependem de conexões estáveis.
